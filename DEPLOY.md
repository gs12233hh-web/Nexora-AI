# 🚀 Guia de Deploy - Nexora AI\n\nEscolha uma das opções abaixo para publicar seu app online!\n\n---\n\n## 1️⃣ **HEROKU** (Recomendado - Mais Fácil)\n\n### Passo 1: Crie conta no Heroku\nhttps://www.heroku.com/\n\n### Passo 2: Instale Heroku CLI\n- **Windows**: Baixe em https://devcenter.heroku.com/articles/heroku-cli\n- **Mac**: `brew tap heroku/brew && brew install heroku`\n- **Linux**: `curl https://cli-assets.heroku.com/install.sh | sh`\n\n### Passo 3: Faça login\n```bash\nheroku login\n```\n\n### Passo 4: Clone o repositório\n```bash\ngit clone https://github.com/gs12233hh-web/Nexora-AI.git\ncd Nexora-AI\n```\n\n### Passo 5: Crie um app no Heroku\n```bash\nheroku create seu-app-nexora\n```\n\n### Passo 6: Deploy\n```bash\ngit push heroku main\n```\n\n### Passo 7: Abra seu app\n```bash\nheroku open\n```\n\n✅ **URL do seu app:** `https://seu-app-nexora.herokuapp.com`\n\n---\n\n## 2️⃣ **VERCEL** (Mais Rápido)\n\n### Passo 1: Crie conta no Vercel\nhttps://vercel.com/\n\n### Passo 2: Instale Vercel CLI\n```bash\nnpm install -g vercel\n```\n\n### Passo 3: Clone o repositório\n```bash\ngit clone https://github.com/gs12233hh-web/Nexora-AI.git\ncd Nexora-AI\n```\n\n### Passo 4: Faça login\n```bash\nvercel login\n```\n\n### Passo 5: Deploy\n```bash\nvercel\n```\n\nSiga as instruções no terminal (escolha o projeto e confirme)\n\n✅ **URL do seu app:** Será exibida após o deploy\n\n---\n\n## 3️⃣ **GITHUB PAGES** (Só Frontend - Sem Backend)\n\n### Passo 1: Ative GitHub Pages\n1. Vá para seu repositório: https://github.com/gs12233hh-web/Nexora-AI\n2. Clique em **Settings** (Configurações)\n3. Vá para **Pages** (no menu esquerdo)\n4. Em \"Source\", selecione **main** branch\n5. Clique em **Save**\n\n✅ **URL do seu app:** `https://gs12233hh-web.github.io/Nexora-AI`\n\n**Obs:** O GitHub Pages só hospeda o frontend (HTML/CSS/JS). O backend não funcionará.\n\n---\n\n## 📊 Comparação\n\n| Recurso | Heroku | Vercel | GitHub Pages |\n|---------|--------|--------|---------------|\n| Frontend | ✅ | ✅ | ✅ |\n| Backend | ✅ | ✅ | ❌ |\n| Banco de Dados | ✅ | ✅ | ❌ |\n| Facilidade | 🟢 Fácil | 🟢 Fácil | 🟢 Muito Fácil |\n| Custo | Grátis* | Grátis | Grátis |\n\n*Heroku oferece horas grátis limitadas\n\n---\n\n## ⚡ Minha Recomendação\n\n**Use HEROKU ou VERCEL** para ter todas as funcionalidades! 🎯\n\n---\n\n## 🆘 Problemas?\n\n**Heroku:**\n- Verifique logs: `heroku logs --tail`\n- Reinicie: `heroku restart`\n\n**Vercel:**\n- Verifique logs no dashboard: https://vercel.com/dashboard\n- Redeploy: `vercel --prod`\n\n---\n\n**Escolha uma opção e boa sorte! 🚀**\n
